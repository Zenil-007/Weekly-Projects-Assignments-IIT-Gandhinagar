
# Clustering Assignment (K-Means, DBSCAN, Hierarchical)

## Overview
This project applies clustering techniques on the Iris dataset without using labels.

## Methods Used
- K-Means
- DBSCAN
- Agglomerative Clustering

## Metrics
- Adjusted Rand Index (ARI)
- Normalized Mutual Information (NMI)
- Silhouette Score

## Key Observations
- K-Means performs well but struggles with overlapping clusters.
- DBSCAN depends heavily on eps parameter.
- Hierarchical clustering provides dendrogram insights.
